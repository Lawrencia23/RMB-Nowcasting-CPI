# RMB Nowcasting CPI Project

This project is part of the Zindi RMB Monthly Mini Challenge.

## Objective
Predict July CPI (Consumer Price Index) using historical CPI and economic indicators.

## Project Structure
- data/ → raw and cleaned datasets
- notebooks/ → EDA and modeling notebooks
- outputs/ → visualizations

## Work Done
- Data cleaning and transformation
- Converted CPI data from long to wide format
- Exploratory Data Analysis (EDA)
- Trend and correlation analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Next Steps
- Feature engineering
- Model building
- Evaluation (RMSE)
